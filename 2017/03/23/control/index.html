&& 与 ||

一、&&

方式：`COMMAND1 && COMMAND2`

如果COMMAND1执行成功，则执行COMMAND2
如果COMMAND1执行失败，则不会执行COMMAND2

二、||

方式：`COMMAND1 || COMMAND2`

如果COMMAND1执行成功，则不会执行COMMAND2
如果COMMAND1执行失败，则执行COMMAND2
